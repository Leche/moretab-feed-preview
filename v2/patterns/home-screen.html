'홈' 칩이 선택된 상태(.screen-home.is-active)에서의 콘텐츠 영역 구성 규칙입니다. 12px 간격의 세로 플렉스 컨테이너 안에 컴포넌트들이 순서대로 쌓입니다.

Released
Gap12px (세로)
Side padding16px

표준 순서

홈 피드의 기본 조합 순서입니다. 섹션 타이틀이 있는 구성의 경우, 타이틀과 카드 사이 간격은 섹션 컴포넌트 자체가 내부에서 처리합니다 (.home-subtitle).

컨테이너 규칙

.screen-home {
  padding: 6px 16px 0;
  display: flex;
  flex-direction: column;
  gap: 12px;
}
.screen-home.is-active { display: flex; }    /* 홈 선택 시만 */
.screen-home:not(.is-active) { display: none; }

/* 섹션 타이틀 */
.home-subtitle {
  font-size: 18px;
  font-weight: 600;
  letter-spacing: -0.3px;
  padding: 18px 0 15px;
}

조합 원칙

1. 상단에서 하단으로 '강 → 약'

사용 빈도·우선순위가 높은 요소가 위에 옵니다. PayStrip과 ServiceShortcut은 반드시 최상단에 배치합니다.

2. 섹션 타이틀은 묶음 단위로

여러 개의 카드가 같은 주제로 묶일 때만 .home-subtitle을 사용합니다. 단일 카드 상단에는 타이틀을 넣지 않습니다.

3. 광고는 섹션 경계에

HomeAd/LocalAd는 콘텐츠 섹션 사이에 배치해 시각적 리듬 변화를 줍니다. 같은 광고 포맷을 연속해서 쓰지 않습니다.
