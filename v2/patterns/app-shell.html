Safe Area 대응

노치·홈 인디케이터가 있는 기종을 위해 상하 safe-area를 CSS 변수로 주입합니다.

:root {
  --safe-top:    env(safe-area-inset-top, 0px);
  --safe-bottom: env(safe-area-inset-bottom, 0px);
}
.top     { padding-top: var(--safe-top); }
.tab-bar { padding-bottom: var(--safe-bottom); }
body     { padding-bottom: 49px; /* tab-bar 만큼 확보 */ }

Background Gradient

화면 전체에 깔리는 .bg-gradient는 콘텐츠 영역에 은은한 톤을 만들어 카드와 배경이 자연스럽게 분리되도록 합니다. 상단 54px과 하단 222px에 각각 화이트 → #f5f5f5 그라디언트가 적용됩니다.

.bg-gradient {
  position: absolute;
  inset: 104px 0 49px 0;  /* header 아래 ~ tabbar 위 */
  background: #f5f5f5;
  z-index: 0;
}
.bg-gradient::before {
  content: ""; position: absolute; top: 0; left: 0; right: 0; height: 54px;
  background: linear-gradient(180deg, #fff 0%, #f5f5f5 100%);
}
.bg-gradient::after {
  content: ""; position: absolute; bottom: 0; left: 0; right: 0; height: 222px;
  background: linear-gradient(0deg, #fff 0%, #f5f5f5 100%);
}
