Color

더보기탭의 기본 색상 시스템입니다. 무채색 8단계의 그레이 팔레트를 중심으로 하며, 특정 용도에만 제한적으로 포인트 컬러를 사용합니다. 모든 색상은 :root의 CSS Custom Property로 정의됩니다.

사용 가이드

Do

  • 본문 텍스트는 --gray900, 보조 정보는 --gray500을 사용합니다.
  • 카드 배경은 #ffffff, 화면 배경은 --gray075 #f5f5f5를 사용합니다.
  • 포인트 컬러는 한 화면 내 2곳을 넘지 않도록 제한합니다.

Don't

  • 포인트 컬러를 장식 목적으로 사용하지 않습니다. 의미가 있는 컨트롤에만 사용합니다.
  • 흐린 회색(--gray300 #b3b3b3 이상 밝은 값)을 본문 텍스트에 사용하지 않습니다. 접근성 대비에 미달합니다.
