시작하기

더보기탭 지면을 새로 설계하거나 기존 섹션을 확장할 때, 아래 순서로 문서를 살펴보면 도움이 됩니다.

1. 토큰 먼저 확인하기

색상, 폰트 사이즈, 여백은 모두 CSS Custom Property로 정의되어 있어, 새로운 요소를 만들 때도 기존 토큰을 우선 사용합니다. Pretendard를 기본 폰트로 사용하고, 본문은 --gray900 / #191919, 보조 정보는 --gray500 / #767676를 사용합니다.

2. 컴포넌트로 조합하기

더보기탭의 홈 피드는 .pay-strip, .service-shortcut, .home-ad, .game-card, .kakaonow-grid, .weather-card 같은 독립 컴포넌트를 세로로 쌓아 올린 구조입니다. 각 컴포넌트의 마진/패딩은 Patterns 페이지를 참고합니다.

3. 패턴으로 화면 구성하기

상단 헤더(104px) — 콘텐츠 영역 — 하단 탭바(49px)로 이뤄지는 앱 셸 구조 안에서 콘텐츠 영역만 변경됩니다. Safe area와 sticky 동작 규칙은 앱 셸 구조에서 확인하세요.

원칙

정보는 카드로, 관계는 간격으로

같은 성격의 정보는 16px 라운드 카드로 묶고, 서로 다른 정보 간의 관계는 12px 섹션 간격으로 구분합니다.

1포인트 컬러 규칙

카카오 옐로우 #fee500는 결제 관련 진입(PayStrip)에만 사용합니다. 그 외에는 그레이스케일을 기본으로, 강조가 필요한 경우 --point-red, --point-blue를 제한적으로 사용합니다.

모바일-퍼스트

기준 뷰포트는 393px(iPhone 15 Pro). 컴포넌트의 내부 패딩은 16px, 카드 간 세로 간격은 12px이 기본값입니다.
